# Пояснительная записка

**Задание**: 

Есть класс трёхмерного вектора:
``` C++
class Vector3D
{
double X;
double Y;
double Z;
...

}
```
и трёхмерного отрезка, заданного двумя Vector3D:

``` C++
class Segment3D
{
Vector3D start;
Vector3D end;
...

}
```
Требуется написать функцию ```Intersect```, которая будет находить точку ```Vector3D``` пересечения двух
заданных на вход ```Segment3D```. В классы ```Vector3D``` и ```Segment3D``` можно добавлять любые методы.

# Среда разработки
Программа написана на C++, в среде разработки Visual Studio 2026.


# Алгоритм решения задачи

Нам дано попарно 4 вектора $`\overline{v}_{00}, \overline{v}_{01}, \overline{v}_{10}, \overline{v1}_{11}`$, образующих 2 сегмента $`s_0, s_1`$. 

Первым шагом рассмотрим частный случай, когда образованные от сегментов пара векторов $`\overline{vs}_0, \overline{vs}_1`$ коллинеарны.
Условием проверки является векторное произведение: $`\overline{vs}_0 \space x \space \overline{vs}_1`$. Для частного случая оно должно быть нулем.
Если условие колинеарности для $`\overline{vs}_0, \overline{vs}_1`$ выполняется, проверяем факт параллельности пары векторов. 
Если они не параллельны, то рассматриваем частные случаи, когда эти векторы касаются друг друга на концах.
В противном случае возвращаем отсутствие решения.

Вторым шагом зададим линейные функции $`F_1(t), \space F_2(u)`$ в параметрическом виде, одразованные от $`\overline{vs}_0, \overline{vs}_1`$ соответственно.

```math
F_1(t) = 
\begin{cases}
x = a_0 t + x_0,\\
y = a_1 t + y_0, \\
z = a_2 t + z_0 \\
\end{cases}
```

```math
F_2(u) = 
\begin{cases}
x = b_0 u + x_1,\\
y = b_1 u + y_1, \\
z = b_2 u + z_1 \\
\end{cases}
```
Введем вектор $`\overline{v}`$ как $`\overline{v} = F_2 - F_1.`$ Поскольку вектор расстояния между 2 прямыми всегда перпендикулярен обоим, то получим систему уравнений

```math
\begin{cases}
\overline{v} \cdot \overline{vs}_0 = 0, \\
\overline{v} \cdot \overline{vs}_1 = 0 
\end{cases}
```
Решив полученное СЛАУ, подставим найденные значения в $`F_1(t), \space F_2(u)`$ и сравним полученные результаты.
