# Пояснительная записка

**Задание**: 

Есть класс трёхмерного вектора:
``` C++
class Vector3D
{
double X;
double Y;
double Z;
...

}
```
и трёхмерного отрезка, заданного двумя Vector3D:

``` C++
class Segment3D
{
Vector3D start;
Vector3D end;
...

}
```
Требуется написать функцию ```Intersect```, которая будет находить точку ```Vector3D``` пересечения двух
заданных на вход ```Segment3D```. В классы ```Vector3D``` и ```Segment3D``` можно добавлять любые методы.

# Алгоритм решения задачи

Нам дано попарно 4 вектора $`\overline{v}_{00}, \overline{v}_{01}, \overline{v}_{10}, \overline{v1}_{11}`$, образующих 2 сегмента $`s_0, s_1`$. Эти сегменты в свою очередь порождают два вектора $`\overline{vs}_0, \overline{vs}_1`$.

Первым шагом определим характер поступивших данных. В программе рассматриваются случаи когда на вход поступили
a. порожденные сегментами две точки (точка в том смысле, что вектор, порожденный сегментом, имеет норму принебрежительно малую;
b. один из порожденных сегментом вектор является точкой;
c. оба вектора колинеарны
d. векторы как минимум скрещивающиеся.

При решении задачи `a.` просто сравниваем координаты двух точек.
При решении задачи `b.` находим расстояние от точки до прямой и если расстояние 0, то устанавливаем -- принаждежит ли точка в необходимому вектору или нет.
При решении задачи `c.` рассматриваем две параметрические функции $`f(t)`$ и $`g(u)`$, основанные на векторах сегментов $`s_0, s_1`$ и устанавливаем лежит ли решение в множестве $`t \in [0, 1] x u \in [0, 1]`$.
При решении последней задачи зададим линейные функции $`F_1(t), \space F_2(u)`$ в параметрическом виде, одразованные от $`\overline{vs}_0, \overline{vs}_1`$ соответственно.

Введем вектор $`\overline{v}`$ как $`\overline{v} = F_2 - F_1.`$ Поскольку вектор расстояния между 2 прямыми всегда перпендикулярен обоим, то получим систему уравнений

```math
\begin{cases}
\overline{v} \cdot \overline{vs}_0 = 0, \\
\overline{v} \cdot \overline{vs}_1 = 0 
\end{cases}
```
Решив полученное СЛАУ, подставим найденные значения в $`F_1(t), \space F_2(u)`$ и сравним полученные результаты. Если они равны и лежат в промежутке $`\in [0; 1]`$, то точка пересечения найдена, иначе ее не существует.
